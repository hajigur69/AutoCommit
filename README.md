# AutoCommit


## Features
- Execute multiple commits simultaneously
- Automate large-scale Git repository updates
- Simplify mass commit workflows

## Quick Start

1. **Clone the repository:**
   ```bash
   git clone https://github.com/hajigur69/AutoCommit.git
   cd AutoCommit
   ```

2. **Run the main script:**
   ```bash
   python3 p.py
   ```

## Usage
```
python3 p.py [options]
```
- Navigate to the repository directory first
- Execute `python3 p.py` to start mass commit operations
- Customize parameters based on your specific needs

## Prerequisites
- Python 3.x installed
- Git installed and configured
- Access to target repositories

## Directory Navigation
```
cd /path/to/your/target/repo
python3 p.py
```

## Common Commands
```
# Basic usage
python3 p.py

# With arguments (check script for supported options)
python3 p.py --help

# Make executable (Linux/Mac)
chmod +x p.py
./p.py
```

## Troubleshooting
- **Permission denied**: `chmod +x p.py`
- **Python not found**: Install Python 3 or use `py p.py` (Windows)
- **Git issues**: Ensure Git is configured with proper credentials
